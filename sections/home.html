DLminibox Documentation

Welcome to the documentation for DLminibox. This library provides functions, modules and optimizers built on top of an automatic differentiation system allowing you to build powerful deep learning AI models. If you haven't already, please start by reading the tutorial to understand the theoretical foundations of this system.

This documentation is unique in the sense each section contains a detailed theoretical explanation following the tutorial, in addition to usage information. I designed this library along with its documentation primarily for teaching purposes, although it is a highly capable system as well.
